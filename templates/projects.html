Spring 2017

CSCI 462 Software Engineering Capstone Project: Mercedes-Benz Vans Plant Visualization Tool

This semester, I worked with 2 student partners to create a web application for Mercedes-Benz Vans using the agile software development method. Our team developed a "Dashboard" solution to monitor all of their IT devices on the production floor. If a device on the floor goes down, the Dashboard map reflects this and production managers are able to pinpoint the problematic device. This software has successfully been installed and is working in the local Mercedes-Benz Vans factory.
